# AccountService - сервис для управления пользовательскими аккаунтами на платформе.

## Описание

Этот репозиторий предоставляет собой сервис по управлению пользовательскими аккаунтами.
Он разработан по чистой архитектуре и использует Identity и Postresql.

Сервис предоставляет проект AccountService.Communication для HTTP взаимодействия с AccountService.
Этот проект опубликован в Nuget.
