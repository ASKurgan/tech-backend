# AccountService - сервис для управления пользовательскими аккаунтами на платформе.

## Описание

Сервис запускается на порту **8001**.

Этот репозиторий предоставляет собой сервис по управлению пользовательскими аккаунтами.
Он разработан по чистой архитектуре и использует Identity и Postresql.

Сервис предоставляет проект AccountService.Communication для HTTP взаимодействия с AccountService.
Этот проект опубликован в Nuget - https://github.com/SachkovTech/sachkov-tech-backend/pkgs/nuget/Accounts.Contracts

Это сервис аутентификации и авторизации, он содержит в себе файл - accounts.json, в котором хранятся все роли и разрешения.

Сервис генирирует key.private и key.pub ключи, генерация происходит, потому что в appsettings в секции "Auth" у ключа "CreateNewKeys" установлено значение true. Пути к ключам указаны там же, в конфигурации.
Эти ключи нужны для подписи и подтверждения подписи у jwt токена. Пара ключей работает следующим образом:
сервис аутентификации один раз создаёт эту пару ключей, когда создается jwt токен, то он подписывается key.private ключом. Для расшифровки используется key.pub ключ. Поэтому key.pub ключ должен находится во всех остальных сервисах в папке etc.
